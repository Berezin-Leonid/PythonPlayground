Problem 1

    Напишите программу, которая принимает на вход число N и
    выводит сумму его цифр.
    
    Example:
        Input: 3143
        Output: 11

        Input: 1111111
        Output: 7

Problem 2

    Напишите программу, которая принимает на вход число N и
    выводит количество четных цифр в этом числе.
    
    Example:
        Input: 3143
        Output: 1 

        Input: 1111111
        Output: 0

        Input: 1234
        Output: 2
